# MERN Practice

Practice repo for learning the MERN stack (MongoDB, Express.js, React, Node.js).

## Project Structure
- `backend/` → server code
- `frontend/` → client code

## How to Run
1. Clone the repo
2. Setup backend:
   ```bash
   cd backend
   npm install
   npm run dev
3. Setup frontend:
  cd frontend
  npm install
  npm start

Tech Stack:
  1. MongoDB
  2. Express
  3. React
  4. Node.js
